# OpenHands Repository

This repository has been structured to improve organization and modularity. Below is an overview of the directory structure:

## Directory Structure

- **`engine/`**: Contains the core `openhands` runner and pipeline logic.
- **`projects/`**: Contains project-specific configurations and pipeline step definitions.
- **`scripts/`**: Contains the executable scripts used to trigger tasks using the `engine`.
- **`apps/`**: Contains standalone applications/servers like `mcp_internet` and `meeting_member_vy`.
- **`plans/`**: Contains documentation and plans for architecture and refactoring.
